## 📖 História do Jogo
A outrora pacífica Vila de Caelum está sob as sombras de uma antiga maldição. Uma masmorra esquecida despertou nas profundezas da terra e o vilão **Z** enviou hordas de monstros para assolar a região. 
Como o último guardião vivo, você deve escolher uma arma sagrada na Vila, adentrar os portões desse labirinto de três andares repletos de perigos e derrotar o Boss Final para trazer a luz de volta ao mundo!

---

## 🎮 Como Jogar

### 🎯 Objetivo
Escolher uma arma na Vila com o NPC, passar pelos 3 andares progressivamente mais desafiadores da masmorra e derrotar o Boss Final (Z) sem perder suas 3 vidas.

### ⌨️ Controles
Os comandos são feitos exclusivamente pelo teclado e não há movimentação na diagonal:
* `W` - Move o jogador para cima (muda o símbolo para `^`)
* `A` - Move o jogador para a esquerda (muda o símbolo para `<`)
* `S` - Move o jogador para baixo (muda o símbolo para `v`)
* `D` - Move o jogador para a direita (muda o símbolo para `>`)
* `I` - Interage com o objeto à frente
* `O` - Realiza o ataque na célula à frente (com base na arma escolhida)

## 🔣 Símbolos do Jogo

| Símbolo | Significado |
| :---: | :--- |
| `< ^ > v` | Jogador e sua direção |
| `*` | Parede (bloqueio) |
| `#` | Espinho (morte instantânea) |
| `k` | Caixa (destruível com ataque) |
| `O` | Botão (ativa mecanismos) |
| `D` / `.` | Porta Fechada / Porta Aberta |
| `@` | Chave (abre portas) |
| `L` | Escada (avança de andar) |
| `X` / `Y` | Monstro Aleatório / Monstro Perseguidor |
| `Z` | Boss Final |
